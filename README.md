# ⏱️ Neo Stopwatch

A modern stopwatch application built with React featuring precise timing functionality.

## LIVE Preview: https://stropwatch-react.vercel.app/

https://github.com/user-attachments/assets/ce88c945-d9b3-4d57-81b9-ac44d9cb287d
## 🚀 Features

- Start stopwatch
- Stop/Pause stopwatch
- Reset stopwatch
- Millisecond precision tracking
- Real-time updates using `setInterval`
- Built with React Hooks
- Stable timer display using formatted values

## 🛠️ Tech Stack

- React
- TypeScript
- CSS
- React Hooks (`useState`, `useRef`)

## 📚 Concepts Practiced

Through this project I learned:

- State management with `useState`
- Persisting mutable values with `useRef`
- Timer handling with `setInterval`
- Managing side effects and cleanup
- Formatting dynamic values
- Conditional rendering

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/saitamacodex/stropwatch-react.git
```

Navigate into the project folder:

```bash
cd stropwatch-react
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
